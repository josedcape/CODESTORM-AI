Asistente de Desarrollo

¡Hola! Soy tu asistente de desarrollo interactivo. Puedo ayudarte a modificar funciones específicas de tu proyecto durante su construcción.

Ejemplos de modificaciones que puedo realizar:
  • Añadir nueva función a un archivo existente: "Agrega una función de validación de email al archivo utils.js"
  • Modificar un componente: "Modifica el componente Botón para que tenga un estado de carga"
  • Optimizar código: "Refactoriza esta función para usar async/await en lugar de promesas"
  • Cambiar estilos: "Actualiza los estilos CSS para que los botones sean más redondeados"
  • Depurar problemas: "Ayúdame a encontrar el error en esta función que causa un TypeError"
Cómo interactuar conmigo para mejores resultados:
  1. Describe claramente qué función o parte del código quieres modificar
  2. Especifica el archivo donde se encuentra (si lo conoces)
  3. Explica el comportamiento actual y el deseado
  4. Si es necesario, activa el Modo intervención para que pueda realizar cambios directos en el proyecto
Cuando activas el Modo intervención, puedo realizar cambios directos en los archivos de tu proyecto. Esto es útil cuando necesitas modificaciones precisas en funciones específicas.
